## Обработка PSD (Class PSDHandle)

Class PSDHandle -- это класс для работы с PSD. Он позволяет: загружать PSD в память, рендерить psd и сохранять итоговый webp.

Class PSDHandle методы:
    - `.render()` -- принимает на вход `{"path_to_psd":{layers for handler}}`. Возращает ссылку на готовый webp для подмены в MD вместо ссылки на `psd`.
    - `.upload()` -- принимает на вход список путей для загрузки в пямять `psd`.
    - `.termanate()` -- удалить из памяти загруженные `psd`.
